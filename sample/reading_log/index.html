편집 안내 — 이 페이지는 정적 카탈로그입니다. 버전 페이지에서 ?edit=1로 켠 편집은 본인 브라우저에만 남고 다른 사람에게는 반영되지 않습니다.

새 버전 공유 절차:
  1. 로컬에서 가장 최근 버전 파일을 ?edit=1로 열고 편집 → 📌 버전 저장으로 다음 버전 HTML 다운로드
  2. 다운로드 파일을 versions/v0.X.html로 이름 변경해 폴더에 추가
  3. versions/manifest.json에 새 버전 항목 추가 (id / file / memo / timestamp / pinCount)
  4. 이 인덱스에 카드 한 개 끼워 넣기 (최신을 맨 위로)
  5. git commit & push → vercel 자동 재배포
