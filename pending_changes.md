# 템플릿 개선 대기 항목

*최종 수정: 2026-04-18*

`spec_template_base.html`에 반영되어야 하는 변경 사항 목록. 처리되면 해당 항목 삭제.

---

## 1. 작성일 표시 필요

- **현재**: `version.timestamp`만 데이터에 존재, 사용자 가시 영역엔 노출 없음
- **변경**: 기획서에 작성일(또는 수정일)을 화면상에 표시
- **미정**: 표시 위치 (상단 헤더? 하단 푸터? 패널 내부?)

## 2. 스펙 패널 동작 변경

- **현재**: 핀 클릭 시 슬라이드로 열리는 방식 (`.spec-panel`)
- **변경**: **항상 노출** + **아코디언** 방식으로 전환
- **미정**: 아코디언 묶음 단위 (핀별? 섹션별 - 역할/상태/속성?)

## 3. 저장 방식 확인 필요

- **현재**: 3단계 저장 모델 (자동 / 💾 작업본 / 📌 버전)
- **변경**: 모델 전반 재검토
- **미정**: 구체적으로 어떤 부분을 재검토할지 — 다음 대화에서 대표님께 먼저 여쭐 것
